# StratologgerCF

The PerfectFlite StratoLoggerCF is a reliable dual deploy altimeter for a very low price of only $65. It is $10 more than the [RRC2+](../missileworks/rrc2.md) however it can log multiple flights in depth and has a computer programmer/flight analysis software. It is a cheaper alternative to the [EasyMini](../easymini.md) and is functionally similar.
