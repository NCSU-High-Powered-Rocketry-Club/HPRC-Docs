# Outreach events

Our club runs a variety of different event types in outreach. This page documents the different events the club currently runs, details how to run them and the resources needed to run each event. 


# Bottle rockets

Bottle rockets is another very fun event we run. This event is most frequently run at outdoor locations often freeflowing in parks or museums. It can however be paired with a presentation as well. 

## Iems needed

- 2 liter soda bottles
- Air compressor
- Bike pump (as backup)
- Aquapod launch stand
- Water cooler (for refilling )
- Table (if not provided)
- Table cloth
- Card board (for fins)
- Tape (for repairs)
- Access to water
- Props
- If presenting: Everything needed from [Presentations](#Presentations)

Please note: Not all of these items will always be needed, for example you will not always need a water cooler, table and tablecloth

## Running the event

This is one of the more tiring events to run, these events are often long, and very interactive with you sometimes launching bottle rockets 100+ times. For bottle rocket events ensure there are at least 4 volunteers to help run the event. 

When launching bottle rockets, have kids pull on the string while you fill up the launch stand with air pressure. You can also have the kid fill up the bottle with water if it's not too busy.

NOTE: make sure to ALWAYS hold the launch stand when they yank the string. If you are not holding the stand when they pull it can cause the launch stand to tip over and cause the rocket to launch directly at the child.
