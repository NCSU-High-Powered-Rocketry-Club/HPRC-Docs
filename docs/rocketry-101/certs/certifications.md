# Certifications Overview

In order to fly high power rockets (motors above H class impulse), you need to be certified by either [Tripoli](https://www.tripoli.org/){target=_blank} or [NAR](https://narocket.clubexpress.com/){target=_blank} which are the two governing bodies of rocketry. There are three levels of certification, each increasing the size of motors available to use. Each level has slightly different requirements which are outlined in their individual pages, however they all basically require you to independently build and fly a rocket.

H-I motors
## [Level 2](../level-2.md)
