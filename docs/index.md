# HPRC Wiki

Welcome to the **NCSU High-Powered Rocketry Club** Wiki.

This is a *(work in progress)* comprehensive guide of all things rocketry. This is a resource for all to use. If you are new to the club start with [Rocketry 101](rocketry-101/getting-started.md) for a guide to getting started in high power rocketry. All pages are written with safety and best practices in mind. If you are a member of HPRC and interested in helping write the wiki please reach out to Aidan on Slack.
